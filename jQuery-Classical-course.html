古典鋼琴課程


班別特色 :古典鋼琴菁英班


招生對象 :

本班別採個別班專業教學，延聘留外歸國鋼琴主修之碩博士菁英， 招收年滿6歲以上、對鋼琴彈奏有高度興趣且願配合訓練計畫執行並以參加各項音樂比賽或報考音樂班為目標者。


課程內容 :

高度要求學生彈奏之儀態姿勢、手型、學習態度，並依照音樂實驗班的課程分配重點教學， 音階、練習曲、作品，按部就班，即使不是要走音樂這條路，仍能將琴詮釋的有專業水準。


學習目標 :

1. 音樂作品完整詮釋彈奏
2. 參加各項鋼琴比賽
3. 報考各級音樂班、音樂系


樂器需求 :

傳統式鋼琴(家中無法準備鋼琴者請勿報名此班別)



班別特色 :古典鋼琴進階班

招生對象 :

本班採一對一個別班精緻教學，特聘國內各大學院校音樂科系、研究所資歷之本科班專業師資， 招收年滿5歲以上或對鋼琴彈奏有興趣的成人皆可，配合老師按部就班的學習計畫下，可以學習鋼琴基礎演奏的技巧與能力。


課程內容 :

以傳統古典鋼琴的教學必修課程與教材，著重於視譜、樂理、視奏節奏等練習， 並以鋼琴古典小品為學習目標，在輕鬆快樂的學習氣氛中，完成鋼琴演奏的能力、培養古典音樂的素養。


學習目標 :

1. 音樂基本能力養成 2.古典鋼琴小品演奏能力 3. 定期參加教室舉辦之檢定、上台演出


樂器需求 :

傳統鋼琴 / 數位鋼琴 (家中若無法購置標準練習工具者，本中心即建議輔導學習其他樂器科類)



班別特色 :古典鋼琴啟蒙班


招生對象 :

本班採一對一個別班愛心、耐心、活潑教學，由國內音樂科系具幼兒教育學程專長或資深幼兒音樂教育講師擔任指導老師， 招收年滿4歲以上的幼稚園小朋友，依每個寶貝的個性與學習能力差異給予量身設計的教學課程。


課程內容 :

透過各種教具的活用，讓孩童感受音樂所包含的節奏快慢強弱、旋律的高低長短、音樂表情的喜怒哀樂並慢慢導入樂譜中的各符號所代表的故事與意義， 並慢慢導入對應到鋼琴鍵盤中的位置，為音樂學習打下良好深厚的基礎。


學習目標 :

節拍、節奏、符號、視譜、鍵盤位置、音樂賞析並引發對音樂的高度興趣。


樂器需求 :

傳統鋼琴 / 數位鋼琴 (家中若無法購置標準練習工具者，本中心即建議輔導學習其他樂器科類)


家長配合：

每週穩定上課不缺課
幼稚園階段家長可陪課
中低年級學生建議家長於老師下課前進入教室瞭解進度與回家練習重點
定時與櫃檯或班主任討論學習進度檢討學習方向或調整師資
每天陪同孩子定時練琴執行老師功課的習慣


班別特色 :古典流行班


招生對象 :

本班招生對象為成人、國小高年級、國中生對鋼琴有興趣但礙於課業壓力無法有太多時間練琴者
覺得傳統古典鋼琴太單調與枯燥乏味,想彈輕鬆之古典小品與流行音樂者
想輕鬆快樂且無負擔的享受學音樂的樂趣者


課程內容：

傳統的古典鋼琴，已經無法滿足現代小朋友的需求了。「古典」與「流行」同時併進的上課方式我們稱之為古典流行班。 練習的曲目大都是「耳熟能詳」的流行歌曲及古典小品，彈奏起來較有成就感，故而較能激勵人心向上學習。 在一堂課內依照小朋友的音樂性向，調整上課內容與方式，讓小朋友快樂地學習，且迫不及待地想回家練習。


教材重點：

1. 快樂學鋼琴教本
2. 快樂學拜爾系列
3.最喜愛的古典小品系列
4.流行豆芽譜
5.理查克萊德曼全集
6.流行爵士鋼琴五線譜版


成果目標：

讓學生不排斥音樂,讓學生會喜歡練琴
讓學生會選自己喜歡的歌曲來彈奏
讓音樂是學生抒壓的興趣而非討厭的才藝補習功課


家長配合：

每週穩定的上課
平時讓學生自己多聽音樂（不限類型）
每天讓學生在課業休息時間保持30分鐘的練習，放鬆身心
定時與櫃檯或班主任討論學習進度檢討學習方向


樂器需求：

傳統鋼琴或數位鋼琴（重觸鍵型）均可

可買練琴卡（1000元/10小時）至教室租琴房練習，每次以一小時為單位
可租鋼琴回家（月租1200~1600元及押金），月租費可抵購琴費用
買中古琴：低價位約3-5萬、中價位約6-8萬、高價位約9--12萬
買數位鋼琴，依型號與功能有不同的價位
